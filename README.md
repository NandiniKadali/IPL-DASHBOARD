This dashboard shows IPL data from 2008 to 2025 in an easy and interactive way.

Season Filter:
You can select any IPL season to see data for that year.
Title Winners:
Shows which teams have won the most IPL titles.
Toss Chart:
Displays how many teams chose to bat or field after winning the toss.
Man of the Match:
Shows top players who won the most Man of the Match awards.
Match Winners:
Shows how many matches each team has won.
Top Venues:
Displays stadiums where most matches were played.

Tools Used:
Excel, Pivot Tables, Pivot Charts, Slicers, and formulas.

Purpose:
To quickly understand IPL team performance, player achievements, and match trends.
